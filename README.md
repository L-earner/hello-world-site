## 🚀 Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/L-earner/hello-world-site.git
   cd hello-world-site
   ```

2. **Open in browser**
   ```bash
   # Simply open index.html in your browser
   open index.html
   ```

3. **Or serve locally**
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js
   npx serve
   
   # PHP
   php -S localhost:8000
   ```
